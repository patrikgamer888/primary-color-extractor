API Usage

You can use this color extractor as an API to get the primary color of any image.

Endpoint

GET /api/extract-color?imageUrl=https://example.com/image.jpg

Or using POST

POST /api/extract-color
Content-Type: application/json

{
  "imageUrl": "https://example.com/image.jpg"
}

Response Format

{
  "imageUrl": "https://example.com/image.jpg",
  "primaryColor": {
    "rgb": "rgb(123, 45, 67)",
    "hex": "#7b2d43",
    "hsl": "hsl(340, 45%, 33%)"
  }
}

Example with cURL

curl -X GET "https://your-netlify-site.netlify.app/api/extract-color?imageUrl=https://example.com/image.jpg"

Example with JavaScript fetch

fetch('https://your-netlify-site.netlify.app/api/extract-color?imageUrl=https://example.com/image.jpg')
  .then(response => response.json())
  .then(data => console.log(data));
